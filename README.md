SkadiBot(浊心斯卡蒂bot)
=
SkadiBot是 基于 ZeroBot 的 明日方舟机器人

使用方式:
-
[1] 部署go-cqhttp https://github.com/Mrs4s/go-cqhttp  
[2] 根据目标机器下载 Releases 中的可执行文件  
[3] 将go-cqhttp配置为正向连接，连接地址和端口与SkadiBot一致  
[4] go-cqhttp中配置qq帐号  
[5] 运行go-cqhttp，并扫码登陆  
[6] 运行SkadiBot
